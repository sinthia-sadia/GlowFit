#  GlowFit – Full-Stack Fitness Tracker Web Application

GlowFit is a full-stack fitness tracking web application designed to help users monitor workouts, track nutrition, manage goals, and maintain consistency in their fitness journey.

The application integrates a dynamic frontend with a backend server and database to provide real-time tracking, persistent storage, and personalized user data management.

# Features

🔐 User Authentication (Register / Login)

🏋️ Workout Logging & History Tracking

🎯 Goal Management System

🥗 Nutrition Tracking

💧 Water Intake Tracker

📊 Interactive Dashboard

📅 Weekly Login Streak Calendar

🧮 BMI Calculator

📱 Fully Responsive Design

☁️ Persistent data storage using MongoDB

Workout & nutrition records are stored and retrieved dynamically
